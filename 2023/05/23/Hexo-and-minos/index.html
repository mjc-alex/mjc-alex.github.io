About Hexo

21.5.23, 我建立了我的第一个（正式的）博客，用的是Github + Hexo. 开头很顺利，今天换主题的时候有些小bug。在此记录过程，方便以后迁移。

  • install node.js, git, hexo

  • connect your github

    open github, click settings > SSH and GPG keys, create a new ssh key.

    copy the output to the new ssh key.

  • modify blog/_config.yml
    open _config.yml, change the last line:

Write new blog

Then open blog/source/_posts, there will be a new file named title.md

If needed to upload images, modify _config.yml:
post_assert_folder: true

Delete blog

delete blog/source/_posts/delete.md, and run hexo d -g
